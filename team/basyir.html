External behavior

Start of Extract [from: User Guide]

Removing a tag from every person : removetag, rt

Removes the specified tag from the address book.
Format: removetag TAG

  • Removes the tag TAG from the address book.

  • The removetag function is case-sensitive.

End of Extract

Justification

User had to delete the entry if they had keyed in a wrong tag and remade a whole other entry with the same parameters to change the tags.

This implementation enhances the usability of the application by allowing the user to update necessary tags and removing them when it becomes obsolete.

Implementation

Start of Extract [from: Developer Guide]

Remove Tag mechanism

The removetag command is facilitated by the RemoveTag class and is an UndoableCommand. RemoveTag checks for the specified tag from all the entries and calls the relevant CommandResult. The specified tag is then removed from the AddressBook.
